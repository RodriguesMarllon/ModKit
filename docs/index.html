ModKit is purpose-built for engineers working with real Modbus hardware. No browser wrapper, no runtime — just Swift and Network.framework.

Native performance
Built entirely in Swift using Network.framework. No Electron, no Node.js — starts instantly and uses minimal resources.
Watch tab
Pin named registers and bits for at-a-glance monitoring. Set scale factors, drag to reorder, and edit watches in-place.
Modbus simulator
Turn your Mac into a Modbus TCP slave. 2,000 holding registers, configurable values — ideal for testing SCADA without real hardware.
Scale factors
Raw Modbus integers often need ÷10, ÷100 to show real engineering values. Apply scale divisors per tag — presets for common ratios.
FC01–FC10 support
Read Coils, Discrete Inputs, Holding Registers, Input Registers. Write single and multiple registers. Binary bit view for every register.
Zero setup
No install scripts, no package managers. Download, unzip, open. IP, port and Unit ID are saved between sessions automatically.

Scanner

Read any register, live.

Connect to any Modbus TCP device and scan a register range with a single click. Auto-poll keeps values fresh at your chosen interval.

Get started

Up in 30 seconds.

ModKit is ad-hoc signed — no App Store, no notarization fees. macOS will ask you to confirm on first launch.

01

Download

Grab the latest .zip from the GitHub Releases page.

02

Unzip & move

Unzip and drag ModKit.app to your /Applications folder.

03

Allow in Security

First launch: go to System Settings → Privacy & Security and click Open Anyway.

$ xattr -cr /Applications/ModKit.app Run this if macOS says "ModKit is damaged"
